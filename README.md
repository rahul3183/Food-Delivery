# Food-Delivery
A food delivery app built with React Native, Sanity.io.

<h3>Techology Stack</h3>
<ul>
<li>React Native</li>
<li>Tailwind CSS</li>
<li>Sanity.io</li>
 </ul>
<br>
  <h3>Dependencies</h3>
  <ul>
  <li>React - UI</li>
  <li>Redux- State Management</li>
  <li>Sanity.io - Store Backend</li>
  <li>Axios - HTTP Request</li>
  <li>Vector Icons - icons</li>
  <li>Tailwind - Component Styling</li>
  <li>Lottie - Animated icons</li>
  </ul>
